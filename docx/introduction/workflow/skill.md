# 流程操作技巧

1. 复制

    在流程标题栏点击右键，会弹出以下右键菜单。点击`复制`即可复制流程。

![image](./images/06.png ':size=90%')

2. 重命名
   
    在流程标题栏双击左键即可打开以下窗口。输入新的流程名称，点击`确定`即可。

![image](./images/07.png ':size=90%')

3. 支持的快捷键 
   
    `Alt + Del` 删除出口连接线

    `Ctrl + C` 复制动作

    `Ctrl + V`粘贴动作

    `Delete` 删除动作

    `Esc` 关闭打开的窗口
