# 网址 
创建一个 *网址* 类型。

![TypeUrl](./images/04.png ':size=90%')

## 子流程
> 不支持


## 运行参数


* 值
> 网址， 例如：`https://winui.net` , 注意要以 `http` 或者 `https` 开头。

## 输出

> 网址，参考：[Url](./types/Url.md)
    


## 脚本调用

```python
import simple

url = simple.Url('https://winui.net')
```
