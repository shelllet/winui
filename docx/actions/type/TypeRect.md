# 矩形 
定义一个 *矩形* 类型。

![TypeRect](./images/09.png ':size=90%')

## 子流程
> 不支持


## 运行参数


* 矩形
> 包含矩形的 *左上角* 坐标和 *长*、 *宽* 大小。
## 输出

> 短形，参考：[Rect](./types/Rect.md)。

## 脚本调用

```python
import simple;

rc = simple.Rect(0,0, 100, 100)

```
