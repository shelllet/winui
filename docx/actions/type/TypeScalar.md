# 标量 
定义 *标量* 类型，*标量* 包含 *4* 个数值。


![TypeScalar](./images/11.png)

## 子流程
> 不支持


## 运行参数


* 值
> 包含四个数值 `M0`、 `M1`、 `M2`、 `M3`。

## 输出

> 标题，参考：[Scalar](./types/Scalar.md)。
    


## 脚本调用

```python
import simple;

s1 = simple.Scalar()

s2 = simple.Scalar(0,0,0,0)

```
