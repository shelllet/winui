# 快捷键 
定义一个 *快捷键* 类型。

![TypeString](./images/08.png ':size=90%')

## 子流程
> 不支持


## 运行参数


* 键
> 快捷键。

## 输出

> 快捷键，参考：[KeySequence](./types/KeySequence.md)。    


## 脚本调用

```python
import simple;

key = simple.KeySequence(simple.Key.Control, simple.Key.W)


```
