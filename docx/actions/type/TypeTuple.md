# 元组 
定义一个 `Tuple`（元组） 类型。

![TypeTuple](./images/17.png ':size=90%')

## 子流程
> 支持，可以通过子动作为该元组添加多个元素。


## 运行参数

* 值
> 元组的第 `1` 个元素。

## 输出

> 元组，参考：[`Tuple`](./types/Tuple.md) 。


## 脚本调用

```python
num = (7, 14, 21, 28, 35)
```
