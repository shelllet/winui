# 指针 
定义一个 指针 类型的值，用来表示内存地址。

![TypePtr](./images/22.png ':size=90%')

## 子流程
> 不支持


## 运行参数

* 值

> 使用字符串或者十六进制官字行串表示一个内存地址，例如：`0x014B30A8`。


## 输出

> 指针    


## 脚本调用

```python
import simple;

ptr = simple.Ptr(0x014B30A8)

```
