# 字符串列表 
定义一个 *字符串列表* 类型的值。

![TypeStrings](./images/21.png ':size=90%')

## 子流程
> 不支持


## 运行参数

* 值
> 多行字符串，注意不是 *Python* 语句或者表达式。执行时按照换行符`\r\n` 分隔为字符串列表。


## 输出

> 字符串列表，参考: [Strings](./types/String.md)。    


## 脚本调用

```python

fruits = ['orange', 'apple', 'pear', 'banana', 'kiwi', 'apple', 'banana']

```
