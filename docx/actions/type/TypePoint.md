# 坐标 
定义一个 *坐标* 类型。

![TypePoint](./images/05.png ':size=90%')

## 子流程
> 不支持


## 运行参数

* 值
> 坐标点，包含 `x` 和 `y` 的值。


## 输出

> 坐标，参考：[Point](./types/Point.md)。
    


## 脚本调用

```python
import simple;

pt = simple.Point(100, 100)

```
