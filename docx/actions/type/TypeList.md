# 列表 
定义一个 *列表* 类型的值。

![TypeList](./images/20.png ':size=90%')

## 子流程
> 不支持


## 运行参数

* 值
>  *Python* 列表， Python 语句，用于创建列表。


## 输出

> 列表，参考: [List](./types/List.md)。    


## 脚本调用

```python

fruits = ['orange', 'apple', 'pear', 'banana', 'kiwi', 'apple', 'banana']

```
