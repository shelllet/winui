# 动作链.发送文本 
向当前有焦点的 *Web* 元素发送文本，参考：[设置文本](./actions/WebElementSendText.md)。

* 该动作需要加入到动作链当中。

![WebActionSendText](./images/22.png ':size=90%')


## 子流程
> 不支持


## 运行参数

* 元素
>   *Web*元素。

* 文本
> 设置的字符串。


## 输出
> 无

* 流程：https://github.com/shelllet/WinUi/blob/main/web/action_sendtxt.simple
