# 判断 
条件判断，用来判断 `True` 或 `False`，也可用来判`空`。相当于`python` 中的 `if`语句，该动作有两条分支，如果条件成立，执行左边的流程分支，否则，执行右边的流程分支。


![If](./images/01.png ':size=90%')

## 子流程

> 不支持

## 运行参数

* 数据
> 可判断的对象,如*Boolean* 类型 。


## 输出

> 是否为空，参考：[Boolean](./types/Boolean.md)， 条件成立流程走左边分支，否则走右侧分支。
