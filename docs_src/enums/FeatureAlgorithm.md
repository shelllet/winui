# FeatureAlgorithm

* SIFT
> 用于检测和描述图像中的局部特征。它对图像缩放、方向和仿射失真具有鲁棒性。在不同尺度的图像中 *SIFT* 都比 *SURF* 要好?
* SURF
> *SURF* 在旋转不变性、模糊和扭曲变换方面优于 *SIFT*?
* BRISK
> ?SURF* 快的一种检测方法?
* ORB
> 计算速度快，并且对光照和旋转变化具有鲁棒性，*ORB* ?*SURF* ?*SIFT* 快得多，而且性能也比 *SURF* 更好?
