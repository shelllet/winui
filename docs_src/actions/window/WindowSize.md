# 窗口大小 
获取或设置窗口的大小（长和宽）?

![WindowSize](./images/13.png ':size=90%')

## 子流?
> 不支?

## 运行参数

* [window](/types/Wnd.md)
  > 窗口
* [size](/types/Size.md)
  > 长宽,如果设置为空，可以获取窗口大小。如果设置长宽，则可以设置窗口的大小?


## 输出

> 窗口尺寸：[`Size`](/types/Size.md), 由于 *Window* 系统窗口的奇葩逻辑，该尺寸不一定等于窗口所设置的尺寸?
